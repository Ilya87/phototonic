Phototonic

Image viewer and organizer

About

Phototonic is an image viewer and organizer for Linux written in C++ and Qt.

Features

  • Light and fast
  • Does not depend on any desktop environment
  • Supports several thumbnail layouts
  • Thumbnails customization: size, spacing, sorting, colors
  • Basic navigation and file management
  • Slide show
  • View random image
  • Transformation: rotation, flipping, cropping
  • Extensive automatic and manual zoom options
  • Keep transformations and zoom while viewing multiple images
  • Multiple mirroring options
  • Supported formats: BMP, GIF, JPEG, PNG, TIFF, SVG, PBM, PGM, PPM, XBM, XPM
  • Supports saving in formats: BMP, JPEG, PNG, TIFF, PPM, XBM, XPM
  • Keyboard shortcuts customization
  • Load image files or folders from command line

    Dependencies

  • Qt 4.x

    • Installation

    In some distributions installation of development packages might be needed beforehand.
    For example, in Ubuntu:

    $ sudo apt-get install build-essential
$ sudo apt-get install libqt4-dev
    General instructions: 
    $ tar -zxvf phototonic.tar.gz
$ cd phototonic
$ qmake-qt4 PREFIX="/usr"
$ make
$ sudo make install
