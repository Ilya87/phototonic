Phototonic

Image viewer and organizer

About

Phototonic is an image viewer and organizer, developed on GNU/Linux in C++ and Qt.

    • Features

  • Light and fast, clear interface
  • Does not depend on any desktop environment
  • Supports several thumbnail layouts
  • Thumbnails customization: size, spacing, sorting, colors
  • Basic navigation and file management
  • Slide show
  • View random image
  • Transformation: rotation, flipping, cropping
  • Extensive automatic and manual zoom options
  • Keep transformations and zoom while viewing multiple images
  • Multiple mirroring options
  • Supported formats: BMP, GIF, JPEG, PNG, TIFF, SVG, PBM, PGM, PPM, XBM, XPM, TGA
  • Supports saving in formats: BMP, JPEG, PNG, TIFF, PPM, XBM, XPM
  • Supports GIF animations
  • Keyboard shortcuts customization
  • Load image files or folders from command line
  • Open image with an external application
  • Change hue and saturation
  • Color splash effect

    Dependencies

  • Qt 5.x

    • Optional:

  • qt5-imageformats (for TIFF and TGA support)
  • qt5-svg (for SVG support)

    • Installation

    General instructions: 
    $ tar -zxvf phototonic.tar.gz
$ cd phototonic
$ qmake PREFIX="/usr"
$ make
$ sudo make install

    In some distributions installation of development packages might be needed prior to installation.
    For example, in Ubuntu:

    $ sudo apt-get install build-essential
$ sudo apt-get install libqt4-dev

    FAQ

    1. When I copy files, why doesn't Phototonic asks whether to overwrite files at the destination?

      2. Phototonic never overwrites files at the destination folder when copying, it always renames the files automatically if it finds files with the same name.

    2. After the upgrade to Qt5, why does the fonts look blurry?

      3. To fix anti aliased fonts in Qt5 applications, edit the file ~/.config/fontconfig/fonts.conf with this content: 

      
<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
<match target="font" >
<edit mode="assign" name="hinting" >
<bool>true</bool>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="hintstyle" >
<const>hintfull</const>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="rgba" >
<const>rgb</const>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="antialias" >
<bool>false</bool>
</edit>
</match>
<match target="font">
<edit mode="assign" name="lcdfilter">
<const>lcddefault</const>
</edit>
</match>
<match target="pattern">
<edit name="dpi" mode="assign">
<double>96</double></edit>
</match>
</fontconfig>
