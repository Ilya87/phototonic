Phototonic

Image viewer and organizer

About

Phototonic is an image viewer and organizer, developed on GNU/Linux in C++ and Qt.

    • Features

  • Light weight, has a clear interface
  • Does not depend on any desktop environment
  • Supports several thumbnail layouts
  • Thumbnails customization: size, spacing, sorting, colors
  • Image navigation and file management
  • Slide show
  • View random image
  • Transformation: rotation, flipping, cropping
  • Extensive automatic and manual zoom options
  • Keep transformations and zoom while viewing multiple images
  • Multiple mirroring options
  • Supported formats: BMP, GIF, JPEG, PNG, TIFF, SVG, PBM, PGM, PPM, XBM, XPM, TGA, ICO
  • Supports saving in formats: BMP, JPEG, PNG, TIFF, PPM, XBM, XPM
  • Supports GIF animations
  • Keyboard shortcuts customization
  • Load image files or folders from command line
  • Open image with an external application
  • Adjust hue and saturation

    Dependencies

  • Qt 5.x

    • Optional:

  • qt5-imageformats (for TIFF and TGA support)
  • qt5-svg (for SVG support)

    • Installation

    General instructions: 
    $ tar -zxvf phototonic.tar.gz
$ cd phototonic
$ qmake PREFIX="/usr"
$ make
$ sudo make install
    Example for Ubuntu: 
    $ sudo apt-get install build-essential
$ sudo apt-get install qt5-default
$ tar -zxvf phototonic.tar.gz
$ cd phototonic
$ qmake qt=qt5
$ make
$ sudo make install

    FAQ

  • When I copy files, why doesn't Phototonic asks whether to overwrite files at the destination?

    • Phototonic never overwrites files at the destination folder when copying, it always renames the files automatically if it finds files with the same name.

  • After the upgrade to Qt5, why does the fonts look blurry?

    • To fix anti aliased fonts in Qt5 applications, edit the file ~/.config/fontconfig/fonts.conf with this content: 

    
<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
<match target="font" >
<edit mode="assign" name="hinting" >
<bool>true</bool>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="hintstyle" >
<const>hintfull</const>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="rgba" >
<const>rgb</const>
</edit>
</match>
<match target="font" >
<edit mode="assign" name="antialias" >
<bool>false</bool>
</edit>
</match>
<match target="font">
<edit mode="assign" name="lcdfilter">
<const>lcddefault</const>
</edit>
</match>
<match target="pattern">
<edit name="dpi" mode="assign">
<double>96</double></edit>
</match>
</fontconfig>
